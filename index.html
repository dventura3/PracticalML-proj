Practical Machine Learning - Prediction Assignment Writeup

The goal of the project was to determine what activity an individual perform using a data set provide by Groupware.

Here, I discuss and compare the results from the training and testing sets using confusion matricies and statistics. A more detailed explaination of the R-Code and how it works can be found in the .Rmd file in this repository. Accuracy of training compared to testing is 1.0 to 0.9951 indicating training for test prediction are accurate. Sensitivity are 1.0 to 0.9993 meaning that the model predictions predict "Classe" membership nearly 100% of the time. Specificity are 1.0 to 0.9986 that the model predictions predict "Non-Classe" membership nearly 100% of the time. In conclusion, the model is well-trained and is very accurate.
